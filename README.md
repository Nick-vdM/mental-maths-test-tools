# Maths Testing Tools
While most websites are enough for me to practice mental maths,
I wanted to be able to fully customise what I can do with it. That's
the point of this.
## maths_test_tools.py
Contains all the functions to run a question
## addition.py
Runs the addition function over and over to practice with
## custom.py
Runs a random function in maths_test_tools.py over and over
